Settings panel

This panel holds the main settings of the application. It lets the user may configure parameters that will be used for an active session.



Configuration files are stored either inside the SDP-INSTALL_DIR/etc/ or inside the user's folder ~/.sdp/etc/ .

At startup, the application will read in priority the user's configuration file if found or have a go at the default one inside the installation folder.


Notice that when a binary hasn't been found, the corresponding line is highlighted. The user may modify the value of the variable at runtime by editing directly the row field.


  • Threads let the user specify the maximum number of jobs to perform at one time
  • Location let the user specify the default location of origins sent into the SeisComP3 system.
  • Detection output let the user configure and apply a filter on generated seismograms snapshots.
  • Monitor let the user specify the maximum space that the root dir containing jobs runs should take and also the minimum size required in order to perform tasks.
  • Cache informs the user of the current application cache size.
