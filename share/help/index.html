SeismicDataPlayback documentation

Welcome into the help section of SeismicDataPlayback.

SeismicDataPlayback (SDP) is a free software registered under the GNU General Public Licence 3. It has been created in order to help process seismological data by setting up and running miscellaneous scripts.
Dispatch scripts are written in bash, and require msrouter and/or msmod (which are compiled by the installer) to route data into a SDS (SeisComP Data Structure).
Detection scripts are written in python and require python-obspy package. (Python 2.7 at least)
Triggers resulting from detections can then be imported into a SeisComP3 system through SeisComP3's scdispatch application.

The application visual interface is defined by six panels :

  • Settings
  • History
  • Activity
  • Trigger
  • Dispatch
  • Detect
