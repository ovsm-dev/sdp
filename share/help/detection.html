Detection panel

In this panel the user may configure a detection script by setting up various parameters like :

  • the data source,
  • the stream selection,
  • the time window definition,
  • the trigger method to apply,
  • the inventory to use.



Loading stations inventory can be done in three ways :

  • by creating inventory by hand (right click, Add network, etc),
  • by loading an ASCII station summary header file
    e.g. : each line of the file should use the format like so :
    ZU,CAPH,19.7355,-72.1883,17.0,2013-08-2T00:00:00,2013-10-10T23:59:00,HHZ, ,HHN, ,HHE, ,
  • by loading a pure dataless file (bin).
The latter is made possible if the user specifies explicitly the location of rdseed bin (from user $path or from the configuration file).



The generated script, written in python, leans on ObsPy library to fetch, process, analyze the data. Triggers if there are any will be listed, pictures generated, everything loaded in the Trigger panel, waiting for user revision.
