Activity panel

This panel offers a queue job manager. The user may manage job execution, termination, reset and deletion.



The number of jobs that the manager can handle is virtually unlimited (pending the computer's ressources).

Jobs in queue get saved each time they get modified. At startup un-processed jobs will be loaded in queue, ready to be launched.


The queue manager can run in two ways :

  • Run selected jobs only by selecting jobs first and then click the 'Run selected' button. If new jobs are added while the queue is running, they will not be scheduled to run.
  • Run all jobs in queue by just clicking the 'Run' button. If the queue is running, freshly addeds jobs will be scheduled to be executed aswell.


Once a job is started, the user may follow its evolution thoughout time by simply selecting it, its standard output is displayed lively.

If needed, a running job can be stopped at any time. Resultings triggers if there is any will still be created and attached to the job of interest.
